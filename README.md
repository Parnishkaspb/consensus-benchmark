### Основная информация:

Проект создан для тестирования 4-х алгоритмов консенсуса: PoS, PBFT, PoW, DAG

### Запуск

Сначала необходимо поднять docker
```bash
  make start_docker
```

Для запуска тестирования есть 2 команды:  
Быстрый тест проводится 15 секунд

```bash
  make quick_start
```

Стандартный тест около 120 секунд
```bash
  make start
```
